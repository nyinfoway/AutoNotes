    Why doesn’t Auto Notes work on this page?

    • Auto Notes commands does not work on blank tabs or chrome local pages, however it works with every other web pages you visit on internet.
    • In Incognito Mode Auto Notes does not work by default, You would have to specifically allow Auto Notes to work in incognito mode.

    Where do my notes get saved? Are they private or saved externally?

    • Your notes are stored in a database locally in your computer, it is completely private to you.
    • We cache absolutely no customer data on our servers.

    I just installed Auto Notes, and it doesn’t seem to be working on my computer.

    • Refreshing the page and see if it helps. Auto Notes will not work on pages which were launched before its installation.
    • Please let us know at chromeautonotes@gmail.com for any issues with Auto notes.

    Why Auto Notes doesn’t work just after installation?

    • Auto Notes will work from new tab after installation.
    • Refresh the already opened page in order to make Auto notes work in your old page.

    Why Auto Notes does not work in incognito mode?

    Auto Notes requires explicit permission to work in incognito mode. Follow these steps to make it work in incognito mode as well.

    • Right click ‘Auto Notes’ icon and select ‘Manage Extensions...’.
    • Locate ‘Auto Notes’ and check ‘Allow in incognito’
    • Now you should be able to use ‘Auto Notes’ in incognito mode also.

    Is  ‘Auto Notes’ extension up-to-date on my computer? How can I benefit from the latest features of the extension?

    • Right click ‘Auto Notes’ icon and select ‘Manage Extensions...’.
    • Check the Developer Mode check box on top right corner of the window.
    • Click on ‘Update extensions now’ button to get the latest version of extension from chrome web store.

    How often do we add features to Auto Notes?

    • Right click ‘Auto Notes’ icon and select ‘Manage Extensions...’
    • Check ‘Developer mode’ check box
    • Click ‘Update extensions now’ button to update to the latest version of ‘Auto Notes’.

    Whom should I contact to report bugs or request new features to ‘Auto Notes’?

    • Email at chromeautonotes@gmail.com for any issues or suggestions about Auto Notes.

    Do you have a email list/newsletter I can subscribe?

    • No we don’t, but kindly write us at chromeautonotes@gmail.com to let us inform you of any new release.
